### A Full-Stack Application using React , Spring Boot and PostgreSql.

I used;

* **React** as fronted
* **Spring Boot**  as backend
* **PostgreSQL** as database
* **Lombok** to get rid of code clutter
* **Bootstrap** and **Semantic UI** as UI
